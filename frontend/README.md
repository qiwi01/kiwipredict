# Kiwi Predict - Frontend

The frontend application for Kiwi Predict, a professional football prediction platform powered by AI algorithms.

## Features

- **AI-Powered Predictions**: Advanced algorithms using Poisson distribution and statistical modeling
- **Real-time Updates**: Live match data and predictions
- **VIP Access**: Premium predictions with enhanced accuracy
- **Bet Converter**: Convert booking codes between sportsbooks
- **Responsive Design**: Works seamlessly on desktop and mobile devices

## Tech Stack

- **React 18** with Vite for fast development and building
- **React Router** for client-side routing
- **Axios** for API communication
- **React Hot Toast** for notifications
- **Lucide React** for icons
- **Tailwind CSS** for styling
- **Recharts** for data visualization

## Getting Started

### Prerequisites

- Node.js 16+
- npm or yarn

### Installation

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:5173](http://localhost:5173) in your browser

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
